变量是干什么用的?

变量支持更多交互式和动态的仪表板。 您不必在度量标准查询中对服务器或传感器名称等内容进行硬编码，而是可以在其位置使用变量。 变量显示为仪表板顶部的下拉选择框。 这些下拉列表可以轻松更改仪表板中显示的数据。 查看 模板文档 来查看更多信息.
